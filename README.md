# Mfcloud Symfony 2

## Installation

### Step 1: Install [Mfcloud](http://mfcloud.readthedocs.org)

### Step2: Init app

``` bash
mfcloud init symfony [path_to_mfcloud_symfony]
```

### Step3: Start app

``` bash
mfcloud start symfony
```
