# MCloud - Symfony 2

## Installation

### Step 1: Install [MCloud](https://mcloud.io)

### Step 2: Init app

``` bash
mcloud init symfony [path_to_mcloud_symfony]
```

### Step 3: Start app

``` bash
mcloud start symfony
```

### Step 4: Show status

``` bash
mcloud status symfony
```
